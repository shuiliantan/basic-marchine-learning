Q : 当发现标签错误时，值得花费时间取修正这些标签吗？

A：深度学习算法对训练集的随机错误很稳健，只要标记的错误不是那么偏离随机分布，比如只是由于人的偶然错误造成的。如果这些错误是很随机的错误，基本上可以不管，不用花费太多时间取修正。当然，纠正也是没问题，有时候也是很有必要的。但是只要总数据量够大，实际错误标记的label占比不高，也可以不做。

但是深度学习算法对系统错误不稳健，例如：如果一直把狗标记为猫，那么算法最后就会把狗预测为猫

![image-20200522095447703](/Users/shuiliantan/Library/Application Support/typora-user-images/image-20200522095447703.png)

建议：

  • dev test set same distribution

  • 对right和wrong的预测都需要check观察。理解机器犯什么错，探究数据，计算错误率，这样可指导下一步去做什么

  • Train slightly different distribbution from dev/test set
